# Horse Powered 3.1.0

## Minecraft 1.21.11 Update

This release updates Horse Powered to Minecraft 1.21.11 with NeoForge 21.11.

### Changes

- Updated to Minecraft 1.21.11
- Updated to NeoForge 21.11.17-beta
- Updated to ModDevGradle 2.0.131
- Migrated NBT serialization to new ValueInput/ValueOutput API
- Migrated BlockEntityRenderer to new render state system
- Updated rendering code for new RenderTypes class location
- Fixed entity restriction API changes (restrictTo/clearRestriction removed)
- Updated UUID serialization for worker entity persistence
- Updated recipe access API usage

### Technical Changes

- AbstractHorse moved to `net.minecraft.world.entity.animal.equine` package
- ResourceLocation renamed to Identifier
- RenderType methods moved to RenderTypes class
- Level.isClientSide field now accessed via isClientSide() method
- Level.getRecipeManager() replaced with recipeAccess()
- BlockEntityRenderer now uses render state pattern with extractRenderState/submit
- CompoundTag getters now return Optional types
- Direction.byName() now returns nullable Direction
